# Library App

A full-stack web application where users can search for books, borrow/renew them, leave reviews, and manage their library account. The application features a modern React frontend and a Spring Boot backend with secure authentication. There can be users given admin roles through Auth0. Only admins are allowed to add new books to the library and respond to queries asked by other users.

## 📚 Features

- **Book Catalog**: Browse and search through an extensive library of books
- **Borrowing System**: Borrow books and manage your borrowing history
- **Renewal & Returns**: Easily renew borrowed books or return them
- **User Reviews**: Leave comments and rate books out of 5 stars
- **Book Ratings**: View average ratings based on user reviews
- **Secure Authentication**: Auth0 for user authentication
- **User Accounts**: Personalized user profiles and borrowing history
- **Admin Role**: Only admins can add new books to the library and answer user questions
