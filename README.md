# Library App

A full-stack web application where users can search for books, borrow them, leave reviews, and manage their library account. The application features a modern React frontend and a Spring Boot backend with secure authentication.

## 📚 Features

- **Book Catalog**: Browse and search through an extensive library of books
- **Borrowing System**: Borrow books and manage your borrowing history
- **Renewal & Returns**: Easily renew borrowed books or return them
- **User Reviews**: Leave comments and rate books out of 5 stars
- **Book Ratings**: View average ratings based on user reviews
- **Secure Authentication**: OAuth2 authentication powered by Okta
- **User Accounts**: Personalized user profiles and borrowing history

## 🛠 Tech Stack

### Frontend
- **React** (v18.2.0) - UI framework
- **TypeScript** - Type-safe JavaScript
- **React Router** (v5.3.4) - Client-side routing
- **Auth0 **
